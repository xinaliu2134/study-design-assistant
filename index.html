MINIMUM CLINICALLY IMPORTANT DIFFERENCE (MCID)

The MCID is the smallest change in outcome that patients or clinicians consider meaningful. It is critical for sample size calculation — using the MCID as the target difference ensures your study is powered to detect clinically relevant effects, not just statistically significant ones.

METHODS FOR DETERMINING MCID

Anchor-based methods
Compare change scores against an external criterion (patient global rating, clinical milestone). Preferred by regulatory agencies. Requires a suitable anchor with moderate correlation (r ≥ 0.3) with the target measure.
Distribution-based methods
Use statistical properties: 0.5 SD (Norman et al., 2003), Standard Error of Measurement (SEM), or Effect Size thresholds. These provide a lower bound but lack clinical context.
TYPE I AND TYPE II ERRORS
TYPE I ERROR (α)

Rejecting a true null hypothesis (false positive). Concluding there is an effect when there isn't one.

Controlled by setting significance level (α, typically 0.05). With multiple comparisons, use Bonferroni correction: α/k or FDR methods.

TYPE II ERROR (β)

Failing to reject a false null hypothesis (false negative). Missing a real effect.

Controlled by increasing sample size, using a larger α, or targeting a larger effect size. Power (1-β) should be at least 0.80 (80%).
